Cloudflare API token requirements

Deploying MTA-STS requires an API token with the following permissions — Zone Edit alone is not sufficient:

  • Account › Workers Scripts: Edit
  • Zone › DNS: Edit
  • Zone › Workers Routes: Edit

Configure the token under Cloudflare › My Profile › API Tokens and update your DNS provider credentials in Robin.
